## 1. 핵심 이론: 왜 듣기가 먼저인가?

```mermaid
flowchart LR
    A[언어 습득 순서] --> B[🦻 듣기<br> Listening]
    B --> C[🗣️ 말하기<br> Speaking]
    C --> D[📖 읽기<br> Reading]
    D --> E[✍️ 쓰기<br> Writing]

    B --> B1["'침묵기 Silent Period'<br> 아이는 먼저 수천 시간 듣고<br> 준비가 되면 스스로 말한다"]
    C --> C1["억지로 시키지 않아도<br> 충분한 Input이 쌓이면<br> 자연스럽게 Output이 나온다"]
```

### 크라센(Krashen)의 입력 가설 — 유아 영어에 적용

| 이론 | 핵심 개념 | 유아 영어 적용 |
|------|----------|--------------|
| **Input Hypothesis** | 이해 가능한 입력 (i+1) 이 있어야 습득 | 아이 수준보다 살짝 높은 영어 노출 |
| **Silent Period** | 말하기 전 충분한 침묵기는 정상 | 영어로 안 말해도 강요하지 않기 |
| **Affective Filter** | 불안·강요 → 언어 습득 차단 | 즐겁고 편안한 환경이 핵심 |
| **Acquisition vs Learning** | 자연 습득 > 의식적 학습 | 놀이·노래·일상 대화로 자연 습득 |

## 2. 듣기(Listening) 중심 전략

### 2-1. 총체적 듣기 환경 설계

```mermaid
flowchart TD
    L[듣기 환경 3층 구조] --> L1[🌊 배경 노출<br> Background Exposure]
    L --> L2[🎯 집중 듣기<br> Focused Listening]
    L --> L3[🤝 상호작용 듣기<br> Interactive Listening]

    L1 --> L1a["하루 종일 영어 음악·동요 틀기<br> 식사·놀이·청소 중 배경음<br> → 뇌에 영어 음소 패턴 각인"]
    L2 --> L2a["부모 영어 그림책 읽어주기<br> 영어 영상 함께 집중해서 보기<br> → 의미와 소리 연결"]
    L3 --> L3a["부모가 영어로 말 걸기<br> 질문하고 기다리기<br> → 반응하며 언어 처리"]
```

### 2-2. 하루 듣기 시간 배분 (권장)

| 유형 | 방법 | 시간 | 효과 |
|------|------|------|------|
| 배경 영어 음악 | Super Simple Songs, 동요 플레이리스트 | 2~3시간 (틀어놓기) | 리듬·억양 내재화 |
| 집중 영상 시청 | Bluey, Peppa Pig (함께 보기) | 30분 | 맥락+어휘 이해 |
| 그림책 읽어주기 | 부모 소리 내어 읽기 | 20~30분 | 소리-의미-그림 통합 |
| 부모 영어 말 걸기 | 일상 속 영어 대화 | 하루 전체 | 실제 언어 습득 |
| **합계** | | **약 3~4시간** | |

## 3. 한국어 최소화 전략 — OPOL & 영어 몰입

### 3-1. OPOL 전략 (One Parent One Language)

```mermaid
flowchart TD
    O[OPOL 전략] --> O1[부모 1<br> 항상 영어만 사용]
    O --> O2[부모 2<br> 한국어 사용]

    O1 --> O1a["엄마: English Only<br> 아빠: 한국어<br> → 아이는 두 언어 자연 구분"]
    O2 --> O2a["예시<br> Mom: 'Time to eat!'<br> Dad: '밥 먹을 시간이야!'<br> → 같은 상황, 다른 언어"]

    O1 --> NOTE["⚠️ 완벽하지 않아도 됨<br> 70~80% 영어 유지 목표<br> 억지로 100% 강요는 스트레스"]
```

### 3-2. 한국어 사용을 줄이는 실전 방법

```mermaid
flowchart LR
    R[한국어 최소화<br> 실전 전략] --> R1[English Only Zone]
    R --> R2[루틴 영어화]
    R --> R3[물건 라벨링]
    R --> R4[반응은 영어로]

    R1 --> R1a["거실·놀이방을 영어 전용 공간<br> '이 방에서는 영어만'<br> 스티커·표지판 활용"]
    R2 --> R2a["아침 인사부터 취침까지<br> 루틴 표현은 항상 영어<br> 'Brush your teeth!'"]
    R3 --> R3a["집 물건에 영어 라벨<br> 물건 볼 때마다 영어 소리<br> 자연스러운 어휘 노출"]
    R4 --> R4a["아이가 한국어로 말하면<br> 영어로 반응해 주기<br> 한국어 교정 말고 모델 제시"]
```

## 4. 아이가 한국어로 말할 때 대처법

> 💡 **핵심 원칙**: 아이가 한국어로 말하는 것은 **완전히 정상**입니다.
> 교정하거나 혼내지 말고, **영어로 자연스럽게 모델을 보여주세요.**

### 4-1. 상황별 대처 전략

```mermaid
flowchart TD
    K[아이가 한국어로 말함] --> K1{상황 파악}
    K1 --> K2[단어를 몰라서<br> 한국어로 표현]
    K1 --> K3[급하거나 감정적일 때<br> 한국어가 튀어나옴]
    K1 --> K4[영어 말하기가 귀찮을 때]

    K2 --> K2a["✅ 전략: Expansion<br> 영어 단어 알려주고<br> 문장으로 확장해 주기"]
    K3 --> K3a["✅ 전략: Acceptance<br> 감정 먼저 받아주고<br> 차분해지면 영어 다시"]
    K4 --> K4a["✅ 전략: Gentle Reminder<br> 'Can you say it in English?'<br> 강요보다 부드럽게 유도"]
```

### 4-2. 실전 대화 예시

| 상황 | 아이 말 (한국어) | ❌ 잘못된 반응 | ✅ 올바른 반응 |
|------|----------------|--------------|--------------|
| 배고플 때 | "엄마, 배고파!" | "한국어 쓰면 안 돼!" | "Oh, you're hungry! Let's eat! Are you hungry?" |
| 물건 가리킬 때 | "저거 뭐야?" | "영어로 말해봐." | "What's that? That's a butterfly! 나비야, butterfly!" |
| 놀이 중 | "이거 내 꺼야!" | (무시) | "It's yours! Mine! This is mine!" |
| 아플 때 | "머리 아파" | "Headache라고 해봐" | "Oh no, does your head hurt? You have a headache." |
| 질문할 때 | "왜?" | "Why라고 해봐야지" | "Why? Good question! Because~" |
| 도움 요청 | "이거 열어줘" | "Open이라고 해야지" | "Open? Can I open it? Here you go! I opened it!" |

### 4-3. 아이의 영어 말하기 유도 기술

```mermaid
flowchart TD
    T[영어 말하기 유도 기술] --> T1[확장 Expansion]
    T --> T2[메아리 Echoing]
    T --> T3[선택 Choice Offering]
    T --> T4[기다리기 Wait Time]
    T --> T5[모델 보이기 Modeling]

    T1 --> T1a["아이: 'dog!'<br> 부모: '예스! A big dog! The dog is running!'"]
    T2 --> T2a["아이: 'ball!'<br> 부모: 'Ball! Yes, it's a red ball!'"]
    T3 --> T3a["'Do you want the apple<br> or the banana?'<br> → 선택지 주면 말하게 됨"]
    T4 --> T4a["질문 후 5~10초 기다리기<br> 바로 답하지 않아도<br> 아이 생각할 시간 주기"]
    T5 --> T5a["부모가 먼저 영어 모델<br> 'Watch me — I say dog!'<br> 아이는 보고 따라함"]
```

## 5. 말하기(Speaking) 발달 단계별 전략

```mermaid
flowchart LR
    S1[침묵기<br> Silent Period<br> 0~6개월] --> S2[모방기<br> Imitation<br> 6개월~1년]
    S2 --> S3[단어 발화<br> Single Words<br> 1~2년]
    S3 --> S4[구문 발화<br> Phrases<br> 2~3년]
    S4 --> S5[문장 발화<br> Sentences<br> 3년+]

    S1 --> S1a["✅ 강요하지 말 것<br> 듣기만 해도 성공<br> 불안해하지 않아도 됨"]
    S2 --> S2a["✅ 따라하면 칭찬<br> 'Good job!<br> You said cat!'"]
    S3 --> S3a["✅ 단어 확장해 주기<br> 'Cat! Big cat! The cat is cute!'"]
    S4 --> S4a["✅ 문장으로 확장<br> 'More milk' →<br> 'I want more milk, please!'"]
    S5 --> S5a["✅ 자연스러운 대화<br> 질문하고 기다리기<br> 대화 주고받기"]
```

### 5-1. 말하기 유도 활동 실전 예시

| 활동 | 방법 | 유도 영어 | 기대 반응 |
|------|------|----------|----------|
| 🎭 **인형극** | 인형으로 대화, 아이가 인형 역할 | "What does the bear say?" | "Hello! / Roar!" |
| 🃏 **그림카드 게임** | 카드 뒤집어 이름 말하기 대결 | "What is it?" | "Dog! / Apple!" |
| 📸 **사진 보며 말하기** | 가족 사진 보며 묘사 | "Who is this?" | "Daddy! / Baby me!" |
| 🔍 **I Spy** | "I spy something blue!" | "Is it the sky? The cup?" | 추측하며 영어 사용 |
| 🍕 **주문 놀이** | 식당 역할극, 메뉴 주문 | "What would you like?" | "Pizza! / Juice please!" |
| 📖 **이야기 완성하기** | 책 읽다가 멈추기 | "And then what happened?" | 이야기 예측 말하기 |
| 🌈 **색깔 챈트** | 리듬에 맞춰 색깔 말하기 | "Red, red, what is red?" | "Apple is red!" |
| 🎵 **노래 빈칸 채우기** | 노래 중간에 멈추고 기다리기 | "Twinkle twinkle little ___" | "Star!" |

## 6. 영어 그림책 읽어주기 — 효과 & 방법

> **결론: 영어 그림책 읽어주기는 매우 효과적입니다.**
> 단, **어떻게** 읽어주느냐가 핵심입니다.

### 6-1. 그림책이 왜 효과적인가?

```mermaid
mindmap
  root((영어 그림책<br> 읽어주기 효과))
    언어 발달
      새로운 어휘 자연 습득
      문장 패턴 반복 노출
      억양·리듬 내재화
    인지 발달
      이야기 구조 이해
      예측·추론 능력
      상상력 자극
    정서 발달
      부모와 유대감
      책 읽기 즐거운 경험
      영어 = 긍정적 감정
    읽기 준비
      문자 인식 기초
      왼→오른쪽 읽기 방향
      소리-문자 연결 준비
```

### 6-2. 그림책 읽어주기 4단계 전략

```mermaid
flowchart TD
    B1[📚 BEFORE 읽기 전<br> 2~3분] --> B2[📖 DURING 읽는 중<br> 10~15분]
    B2 --> B3[💬 AFTER 읽은 후<br> 5분]
    B3 --> B4[🎭 EXTENSION 확장 활동<br> 10~20분]

    B1 --> B1a["표지 보며 예측<br> 'What do you see?'<br> 'What do you think<br> will happen?'"]
    B2 --> B2a["그림 짚으며 묘사<br> 효과음 넣기<br> 반복 구절 같이 읽기<br> 표정·몸짓 함께"]
    B3 --> B3a["'What happened?'<br> 'Who was your favorite?'<br> 'Did you like it?'"]
    B4 --> B4a["역할극<br> 그림 그리기<br> 이야기 다시 말하기<br> 다른 결말 만들기"]
```

### 6-3. 그림책 읽어줄 때 영어 발화 기술

| 기술 | 방법 | 예시 |
|------|------|------|
| **짚기 (Pointing)** | 그림 짚으며 이름 말하기 | "Look! A caterpillar! He's eating!" |
| **효과음 (Sound Effects)** | 동물 소리, 행동 소리 넣기 | "The bear goes ROAR! 🐻" |
| **질문하기 (Questioning)** | 열린 질문으로 생각 유도 | "Why is she crying? What do you think?" |
| **예측하기 (Predicting)** | 페이지 넘기기 전 멈추기 | "What will happen next? Let's see!" |
| **연결하기 (Connecting)** | 아이 경험과 연결 | "Like when YOU were scared! Remember?" |
| **반복하기 (Repetition)** | 반복 구절은 같이 읽기 | 아이와 함께 "Brown Bear, what do you see?" |
| **드라마틱하게 (Drama)** | 목소리·속도·크기 변화 | 속삭이다가 크게, 천천히 했다가 빠르게 |

### 6-4. 부모 영어 실력이 부족할 때 해결책

```mermaid
flowchart TD
    P[부모 영어 자신 없을 때] --> P1[오디오북 활용]
    P --> P2[YouTube 읽어주기 영상]
    P --> P3[발음 앱 먼저 연습]
    P --> P4[간단한 책부터 시작]

    P1 --> P1a["Audible, Epic Books<br> 원어민 목소리 오디오<br> 함께 들으며 따라 읽기"]
    P2 --> P2a["Storyline Online<br> Read Aloud 채널<br> 원어민이 읽어주는 영상"]
    P3 --> P3a["Google 번역 발음 기능<br> Forvo 발음 사이트<br> 책 읽기 전 먼저 연습"]
    P4 --> P4a["BOB Books, ORT Stage 1<br> 3~5단어 짧은 문장<br> 발음 걱정 없이 시작"]
```

### 7-2. 상황별 부모 영어 스크립트

#### 🌅 아침 (Morning Routine)
```
▶ 기상
"Good morning! Rise and shine! ☀️"
"Did you sleep well? Did you have any dreams?"
"Let's get up! Time to start the day!"

▶ 세수/양치
"Let's wash your face! Splash, splash!"
"Now brush your teeth — top, bottom, sides!"
"Open wide! Good job! All clean!"

▶ 옷 입기
"What do you want to wear today?"
"It's cold today — let's wear a jacket!"
"Put your arms through! There you go! Looking good!"
```

#### 🍳 식사 시간 (Meal Time)
```
▶ 식사 전
"Are you hungry? Let's eat!"
"What do you want for breakfast?"
"We're having eggs and toast today!"

▶ 식사 중
"How is it? Is it yummy?"
"Do you want more? / Are you full?"
"Take a big bite! Chew, chew, chew!"
"Drink some water, please."

▶ 식사 후
"All done! Did you enjoy it?"
"Let's clean up! Wipe your mouth."
"Good eating today! 👍"
```

#### 🎮 놀이 시간 (Playtime)
```
▶ 놀이 시작
"What do you want to play?"
"Let's play with blocks! / Let's do a puzzle!"
"I'll play with you! Let's go!"

▶ 놀이 중
"Wow, look at that! Amazing!"
"Your turn! / My turn!"
"What are you making? Tell me!"
"Be careful! / Good job!"

▶ 정리 시간
"Clean up time! Let's tidy up!"
"Where does this go? On the shelf!"
"Almost done! Great teamwork!"
```

#### 🌙 취침 루틴 (Bedtime Routine)
```
▶ 목욕
"Bath time! Let's go! 🛁"
"Is the water warm? Good!"
"Wash, wash, wash! Now rinse!"

▶ 독서
"Story time! Which book tonight?"
"Snuggle up! Let's read together."
(읽은 후) "The end! Did you like it?"

▶ 잠자리
"Time to sleep. Close your eyes."
"I love you so much! 💕"
"Sweet dreams! Good night!"
```

## 8. 유아 영어 방법론 비교

| 방법론 | 핵심 원칙 | 특징 | 한국 가정 적용 |
|--------|----------|------|--------------|
| **자연습득법 (Natural Approach)** | 언어는 자연스럽게 습득 | 강요 없이 의미 있는 입력 | 일상 영어 대화 |
| **전신반응법 (TPR)** | 신체 동작으로 언어 습득 | 지시 → 행동으로 반응 | "Jump! Clap! Sit!" 명령 놀이 |
| **몰입법 (Immersion)** | 목표 언어만 사용 환경 | 최대 영어 노출 | English Only Zone 설정 |
| **균형적 접근 (Balanced)** | 자연 습득 + 구조적 학습 | 놀이+파닉스 병행 | 4~5세부터 파닉스 추가 |

## 10. ⚠️ 절대 하지 말아야 할 것

```mermaid
flowchart TD
    STOP[🚫 절대 금지 사항] --> S1["❌ 한국어로 말하면 혼내기<br> → 영어 거부감·트라우마"]
    STOP --> S2["❌ 강제로 영어 발화 시키기<br> → 침묵기 무시, 불안 유발"]
    STOP --> S3["❌ 혼자 영상 보게 방치<br> → 수동 시청 = 학습 효과 없음"]
    STOP --> S4["❌ 발음 틀릴 때마다 교정<br> → 자신감 저하, 말하기 거부"]
    STOP --> S5["❌ 다른 아이와 비교<br> → 영어 = 스트레스로 인식"]
    STOP --> S6["❌ 한국어를 완전히 금지<br> → 모국어 발달 저해, 정체성 혼란"]
    STOP --> S7["❌ 하루 2시간+ 영어 영상<br> → 과도한 화면 노출, 수면/발달 문제"]
    STOP --> S8["❌ 너무 어려운 콘텐츠 강요<br> → 이해 안 되면 영어 흥미 상실"]
```
