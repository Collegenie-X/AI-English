# 🧒 유아 150패턴 마스터 테스트

## 📝 테스트 안내

이 파일은 **전설의 유아 회화 150패턴**을 완벽하게 마스터했는지 확인하는 테스트입니다.

### 테스트 구성
- **Part 1**: 한글 → 영어 (50문제)
- **Part 2**: 영어 → 한글 (50문제)
- **Part 3**: 청크 채우기 (30문제)
- **Part 4**: 상황별 응용 (20문제)

### 힌트 시스템
- 🟢 **쉬움**: 첫 글자 제공
- 🟡 **보통**: 단어 수 힌트
- 🔴 **어려움**: 힌트 없음
