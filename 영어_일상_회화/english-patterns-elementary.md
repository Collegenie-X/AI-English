## 학습 흐름도

```mermaid
flowchart LR
    subgraph 기본["🌱 기본"]
        A[인사/소개] --> B[감정/상태]
    end
    
    subgraph 생활["🏠 생활"]
        C[일상] --> D[학교]
    end
    
    subgraph 소통["💬 소통"]
        E[취미] --> F[친구]
        F --> G[요청/제안]
        G --> H[질문/대답]
    end
    
    기본 --> 생활
    생활 --> 소통
    
    style A fill:#e8f5e9
    style C fill:#fff3e0
    style E fill:#e3f2fd
```
