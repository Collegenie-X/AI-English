## 일상생활 테스트

### 📝 테스트 1: 편의점 (5문제)

```
상황: 편의점에서 물건 구매

문제 1: 직원이 "Find everything?"라고 물었습니다.
당신의 답변은?
a) "Yes, thank you."
b) "Yeah, just this."
c) "I'm looking for water."

🔊 발음: "예, 땡큐." / "예, 저스트 디스." / "아임 루킹 포 워러."
⏱️ 제한 시간: 2초

문제 2: 충전기를 찾고 싶습니다. 뭐라고 물어볼까요?
a) "Where is charger?"
b) "Do you have phone chargers?"
c) "Charger please."

🔊 발음: "웨어 이즈 차저?" / "두 유 해브 폰 차저즈?" / "차저 플리즈."
⏱️ 제한 시간: 2초

문제 3: 카드로 결제하고 싶습니다. 뭐라고 말할까요?
a) "Card."
b) "Can I pay with card?"
c) "I want card payment."

🔊 발음: "카드." / "캔 아이 페이 위드 카드?" / "아이 원트 카드 페이먼트."
⏱️ 제한 시간: 1초

문제 4: 영수증이 필요 없습니다. 뭐라고 말할까요?
a) "No receipt."
b) "I don't need receipt."
c) "Nah, I'm good."

🔊 발음: "노 리씨트." / "아이 돈트 니드 리씨트." / "나, 아임 굿."
⏱️ 제한 시간: 1초

문제 5: 화장실이 어디인지 물어보세요.
a) "Bathroom?"
b) "Where's the restroom?"
c) "I need bathroom."

🔊 발음: "배스룸?" / "웨어즈 더 레스트룸?" / "아이 니드 배스룸."
⏱️ 제한 시간: 1초
```

<details>
<summary>✅ 정답 및 해설</summary>

1. **b) "Yeah, just this."** ⭐ 가장 자연스러움
   - a도 OK지만 b가 더 캐주얼하고 자연스러움
   - c는 아직 찾는 중일 때

2. **b) "Do you have phone chargers?"** ⭐ 가장 명확
   - 완전한 문장으로 정중하게
   - a는 문법적으로 부족
   - c는 너무 직접적

3. **b) "Can I pay with card?"** ⭐ 가장 정중
   - a도 OK지만 b가 더 완전한 문장
   - c는 부자연스러움

4. **c) "Nah, I'm good."** ⭐ 가장 자연스러움
   - 현지인이 가장 많이 사용
   - a도 OK
   - b는 너무 격식적

5. **b) "Where's the restroom?"** ⭐ 가장 명확
   - 완전한 질문 형태
   - a도 OK지만 b가 더 정중
   - c는 급한 느낌

**채점:**
- 5개 정답: 마스터! 🏆
- 3-4개: 우수! 👍
- 1-2개: 연습 필요! 💪

</details>

### 📝 테스트 2: 약국 (5문제)

```
상황: 약국에서 감기약 구매

문제 1: 감기약이 필요합니다. 뭐라고 말할까요?
a) "I need cold medicine."
b) "Give me cold medicine."
c) "Cold medicine please."

🔊 발음: "아이 니드 콜드 메디슨." / "기브 미 콜드 메디슨." / "콜드 메디슨 플리즈."
⏱️ 제한 시간: 2초

문제 2: 약사가 "What are your symptoms?"라고 물었습니다.
당신의 증상은 콧물과 목 아픔입니다. 뭐라고 답할까요?
a) "Runny nose and sore throat."
b) "My nose is running and throat hurts."
c) "I have runny nose, sore throat."

🔊 발음: "러니 노즈 앤드 쏘어 쓰로트." / "마이 노즈 이즈 러닝 앤드 쓰로트 허츠." / "아이 해브 러니 노즈, 쏘어 쓰로트."
⏱️ 제한 시간: 3초

문제 3: 졸리지 않은 약을 원합니다. 뭐라고 말할까요?
a) "Not sleepy medicine."
b) "Non-drowsy, please."
c) "I don't want to sleep."

🔊 발음: "낫 슬리피 메디슨." / "논 드라우지, 플리즈." / "아이 돈트 원트 투 슬립."
⏱️ 제한 시간: 2초

문제 4: 음식과 함께 먹어야 하는지 물어보세요.
a) "With food?"
b) "Can I take it with food?"
c) "Do I need to eat?"

🔊 발음: "위드 푸드?" / "캔 아이 테이크 잇 위드 푸드?" / "두 아이 니드 투 잇?"
⏱️ 제한 시간: 2초

문제 5: 얼마인지 물어보세요.
a) "Price?"
b) "How much?"
c) "How much is it?"

🔊 발음: "프라이스?" / "하우 머치?" / "하우 머치 이즈 잇?"
⏱️ 제한 시간: 1초
```

<details>
<summary>✅ 정답 및 해설</summary>

1. **a) "I need cold medicine."** ⭐ 가장 명확
   - 완전한 문장으로 정중
   - b는 너무 직접적
   - c도 OK

2. **a) "Runny nose and sore throat."** ⭐ 가장 간단명료
   - 핵심만 전달
   - b도 OK지만 길다
   - c는 문법적으로 어색

3. **b) "Non-drowsy, please."** ⭐ 전문 용어 사용
   - 약국에서 사용하는 정확한 용어
   - a는 부자연스러움
   - c는 의도가 불명확

4. **b) "Can I take it with food?"** ⭐ 가장 명확
   - 완전한 질문 형태
   - a도 OK지만 b가 더 정중
   - c는 의미가 다름

5. **b) "How much?"** ⭐ 가장 자연스러움
   - 간단하고 명확
   - a는 너무 직접적
   - c도 OK지만 b가 더 간결

**채점:**
- 5개 정답: 약국 마스터! 💊
- 3-4개: 잘하고 있어요! 👍
- 1-2개: 복습 필요! 📚

</details>

## 긴급상황 테스트

### 📝 테스트 3: 분실 신고 (5문제)

```
상황: 지갑을 잃어버렸습니다

문제 1: 호텔 프론트에 지갑을 잃어버렸다고 말하세요.
a) "I lost my wallet."
b) "My wallet is gone."
c) "I can't find my wallet."

🔊 발음: "아이 로스트 마이 월렛." / "마이 월렛 이즈 곤." / "아이 캔트 파인드 마이 월렛."
⏱️ 제한 시간: 2초

문제 2: 언제 잃어버렸는지 물어봅니다. "About 30 minutes ago"라고 답하세요.
어떻게 발음하나요?
a) 어바웃 써티 미닛츠 어고
b) 어바웃 쓰리 제로 미닛츠 어고
c) 서티 미닛츠 비포

🔊 정답 발음 확인
⏱️ 제한 시간: 2초

문제 3: 어디서 잃어버렸는지 모릅니다. 뭐라고 말할까요?
a) "I don't know where."
b) "I'm not sure where I lost it."
c) "Maybe restaurant?"

🔊 발음: "아이 돈트 노우 웨어." / "아임 낫 슈어 웨어 아이 로스트 잇." / "메이비 레스토랑?"
⏱️ 제한 시간: 2초

문제 4: 신용카드를 취소해야 합니다. 뭐라고 말할까요?
a) "I need to cancel my credit card."
b) "Cancel credit card please."
c) "Can you help me cancel my card?"

🔊 발음: "아이 니드 투 캔슬 마이 크레딧 카드." / "캔슬 크레딧 카드 플리즈." / "캔 유 헬프 미 캔슬 마이 카드?"
⏱️ 제한 시간: 3초

문제 5: 경찰 신고가 필요한지 물어보세요.
a) "Police?"
b) "Should I call the police?"
c) "Do I need to report to police?"

🔊 발음: "폴리스?" / "슈드 아이 콜 더 폴리스?" / "두 아이 니드 투 리포트 투 폴리스?"
⏱️ 제한 시간: 2초
```

<details>
<summary>✅ 정답 및 해설</summary>

1. **a) "I lost my wallet."** ⭐ 가장 명확하고 직접적
   - 긴급 상황에서는 명확함이 중요
   - b도 OK
   - c는 아직 찾을 수 있다는 뉘앙스

2. **a) 어바웃 써티 미닛츠 어고** ⭐ 정확한 발음
   - thirty는 "써티"로 발음
   - b는 부자연스러움
   - c는 의미가 약간 다름

3. **b) "I'm not sure where I lost it."** ⭐ 가장 완전한 답변
   - 상황을 명확히 설명
   - a도 OK지만 b가 더 정중
   - c는 추측일 뿐

4. **a) "I need to cancel my credit card."** ⭐ 가장 명확
   - 본인이 해야 할 일을 명확히 표현
   - c도 OK (도움 요청)
   - b는 너무 직접적

5. **b) "Should I call the police?"** ⭐ 가장 자연스러운 질문
   - 조언을 구하는 형태
   - c도 OK
   - a는 너무 간단

**채점:**
- 5개 정답: 긴급 상황 대비 완벽! 🚨
- 3-4개: 잘 준비됐어요! 👍
- 1-2개: 더 연습하세요! 💪

</details>

### 📝 테스트 4: 의료 긴급 (5문제)

```
상황: 갑자기 어지럽습니다

문제 1: 어지럽다고 말하세요.
a) "I'm dizzy."
b) "My head is spinning."
c) "I feel dizzy."

🔊 발음: "아임 디지." / "마이 헤드 이즈 스피닝." / "아이 필 디지."
⏱️ 제한 시간: 1초

문제 2: 앉아야 할 것 같습니다. 뭐라고 말할까요?
a) "I need to sit."
b) "Can I sit down?"
c) "I need to sit down."

🔊 발음: "아이 니드 투 싯." / "캔 아이 싯 다운?" / "아이 니드 투 싯 다운."
⏱️ 제한 시간: 2초

문제 3: 물이 필요합니다. 뭐라고 말할까요?
a) "Water please."
b) "Can I get some water?"
c) "I need water."

🔊 발음: "워러 플리즈." / "캔 아이 겟 썸 워러?" / "아이 니드 워러."
⏱️ 제한 시간: 1초

문제 4: 병원에 가야 하는지 물어보세요.
a) "Hospital?"
b) "Should I go to the hospital?"
c) "Do I need hospital?"

🔊 발음: "하스피털?" / "슈드 아이 고 투 더 하스피털?" / "두 아이 니드 하스피털?"
⏱️ 제한 시간: 2초

문제 5: 구급차를 불러달라고 하세요.
a) "Call ambulance!"
b) "I need an ambulance."
c) "Please call an ambulance."

🔊 발음: "콜 앰뷸런스!" / "아이 니드 언 앰뷸런스." / "플리즈 콜 언 앰뷸런스."
⏱️ 제한 시간: 2초
```

<details>
<summary>✅ 정답 및 해설</summary>

1. **a) "I'm dizzy."** ⭐ 가장 간단명료
   - 긴급 상황에서는 간단함이 중요
   - c도 OK
   - b는 좀 더 문학적

2. **c) "I need to sit down."** ⭐ 가장 명확
   - "sit down"이 완전한 표현
   - b도 OK (허락 구함)
   - a는 "down" 빠짐

3. **b) "Can I get some water?"** ⭐ 가장 정중
   - 긴급하지만 정중하게
   - a, c도 OK
   - 상황에 따라 선택

4. **b) "Should I go to the hospital?"** ⭐ 가장 자연스러운 질문
   - 조언을 구하는 형태
   - a는 너무 간단
   - c는 문법적으로 어색

5. **c) "Please call an ambulance."** ⭐ 긴급하지만 정중
   - 명확한 요청
   - b도 OK
   - a는 너무 직접적 (긴급 시 OK)

**채점:**
- 5개 정답: 의료 긴급 대비 완벽! 🏥
- 3-4개: 잘 준비됐어요! 👍
- 1-2개: 더 연습하세요! 💪

</details>

## 전설의 6턴 시뮬레이션

### 🎬 시뮬레이션 1: 편의점 전체 과정 (8턴)

```
상황: 편의점에서 물과 충전기 구매
목표: 입장부터 퇴장까지 완벽 소화

턴 1 - 직원 → 손님
영어: "Hey! / Find everything?"
🔊 발음: "헤이! / 파인드 에브리띵?"
📝 청크: [인사] + [찾았나요]
💭 직독직해: "안녕하세요 / 다 찾으셨어요"
⏱️ 이해 시간: 0.5초

당신의 답변: _________________
⏱️ 제한 시간: 2초

턴 2 - 손님 → 직원
모범 답안: "Yeah, / just this."
🔊 발음: "예, / 저스트 디스."
📝 청크: [네] + [이것만]
💭 직독직해: "네 / 이것만요"

턴 3 - 직원 → 손님
영어: "Cool. / That's $3.50."
🔊 발음: "쿨. / 댓츠 쓰리 피프티."
📝 청크: [좋아요] + [가격]
💭 직독직해: "좋아요 / 3.50달러예요"
⏱️ 이해 시간: 0.5초

당신의 답변 (카드로 결제): _________________
⏱️ 제한 시간: 1초

턴 4 - 손님 → 직원
모범 답안: "Can I pay / with card?"
🔊 발음: "캔 아이 페이 / 위드 카드?"
📝 청크: [가능한가요] + [카드로]
💭 직독직해: "제가 낼 수 있나요 / 카드로"

턴 5 - 직원 → 손님
영어: "For sure! / Tap or insert?"
🔊 발음: "포 슈어! / 탭 오어 인서트?"
📝 청크: [물론] + [터치? 삽입?]
💭 직독직해: "물론이죠 / 터치하시겠어요 넣으시겠어요"
⏱️ 이해 시간: 1초

당신의 답변: _________________
⏱️ 제한 시간: 1초

턴 6 - 손님 → 직원
모범 답안: "Tap."
🔊 발음: "탭."
📝 청크: [터치]

턴 7 - 직원 → 손님
영어: "Perfect. / Receipt?"
🔊 발음: "퍼펙트. / 리씨트?"
📝 청크: [완벽] + [영수증?]
💭 직독직해: "완벽해요 / 영수증 받으시겠어요"
⏱️ 이해 시간: 0.5초

당신의 답변 (필요 없음): _________________
⏱️ 제한 시간: 1초

턴 8 - 손님 → 직원
모범 답안: "Nah, / I'm good."
🔊 발음: "나, / 아임 굿."
📝 청크: [아니요] + [괜찮아요]
💭 직독직해: "아니요 / 괜찮아요"

턴 9 - 직원 → 손님
영어: "Alright! / Have a good one!"
🔊 발음: "올라잇! / 해브 어 굿 원!"
📝 청크: [알겠어요] + [좋은 하루]
💭 직독직해: "알겠어요 / 좋은 하루 보내세요"

당신의 답변: _________________
⏱️ 제한 시간: 1초

턴 10 - 손님 → 직원
모범 답안: "Thanks!"
🔊 발음: "땡스!"
📝 청크: [감사]

✅ 완료 시간: 1분
🎯 청크 개수: 18개
💪 난이도: 초급
```
