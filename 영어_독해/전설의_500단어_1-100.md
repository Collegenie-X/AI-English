# 전설의 500단어 (001-100) - 문장 패턴으로 마스터

## 🎯 이 파일의 목적

단어 하나하나를 외우는 것이 아니라, **실제 문장 패턴 속에서** 100개 핵심 단어를 익힙니다.
각 단어는 3개의 필수 패턴 문장과 함께 제공되어 **청크 단위로 자동 암기**됩니다.

```mermaid
mindmap
  root((001-100단어))
    구성
      동사 50개
      명사 30개
      형용사 20개
    특징
      최빈출 단어
      일상 필수
      기초 토대
    학습법
      청크 인식
      직독직해
      패턴 반복
```

---

## 📚 학습 방법

### 단계별 학습법

```mermaid
flowchart LR
    A[1단계<br/>패턴 읽기] --> B[2단계<br/>청크 인식]
    B --> C[3단계<br/>직독직해]
    C --> D[4단계<br/>반복 연습]
    D --> E[5단계<br/>테스트]
    
    style A fill:#e1f5ff,color:#111
    style C fill:#ffe1e1,color:#111
    style E fill:#ccffcc,color:#111
```

| 단계 | 방법 | 시간 | 목표 |
|------|------|------|------|
| **1단계** | 패턴 문장 읽기 | 1분 | 3개 패턴 이해 |
| **2단계** | 슬래시로 청크 구분 | 30초 | 청크 인식 |
| **3단계** | 직독직해로 해석 | 1분 | 어순대로 이해 |
| **4단계** | 소리내어 반복 | 2분 | 자동화 |
| **5단계** | 스스로 문장 만들기 | 1분 | 실전 적용 |
