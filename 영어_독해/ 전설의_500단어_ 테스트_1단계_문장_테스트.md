# 전설의 500단어 - 1단계 문장 테스트

## 🎯 1단계 테스트의 목적

500단어를 사용한 **실전 문장**을 읽고 이해하는 능력을 테스트합니다.
단어의 의미를 알고 있는지, 문장 전체를 이해할 수 있는지 확인합니다.

```mermaid
mindmap
  root((1단계<br/>문장 테스트))
    목표
      단어 의미 파악
      문장 이해
      상황 파악
    구성
      기초 50문장
      중급 50문장
      고급 50문장
      비즈니스 50문장
    평가
      의미 이해도
      문맥 파악
      빠른 독해
```

---

## 📝 테스트 방법

### 진행 순서

```mermaid
flowchart LR
    A[문장 읽기] --> B[의미 파악]
    B --> C[질문 답하기]
    C --> D[정답 확인]
    
    style A fill:#e1f5ff,color:#111
    style B fill:#ffe1e1,color:#111
    style C fill:#fff4e1,color:#111
    style D fill:#ccffcc,color:#111
```

**지시사항:**
1. 각 문장을 읽고 의미를 파악하세요
2. 문장 아래의 질문에 답하세요
3. 정답을 확인하고 이해도를 체크하세요

```mermaid
flowchart TD
    A[총 100문제] --> B[맞은 개수 세기]
    B --> C[정답률 계산]
    C --> D{90% 이상?}
    
    D -->|Yes| E[우수: 2단계 진행]
    D -->|No| F{80% 이상?}
    
    F -->|Yes| G[양호: 복습 후 2단계]
    F -->|No| H[부족: 재학습 필요]
    
    style E fill:#gold,color:#111
    style G fill:#ccffcc,color:#111
    style H fill:#ffcccc,color:#111
```

### 점수 기록

**나의 점수:**
- 기초 레벨 (1-50): _____ / 50 (____%)
- 중급 레벨 (51-100): _____ / 50 (____%)
- **총점**: _____ / 100 (____%)

### 평가 기준

| 점수 | 평가 | 다음 단계 |
|------|------|-----------|
| **90-100%** | 우수 | 즉시 2단계 진행 |
| **80-89%** | 양호 | 틀린 부분 복습 후 2단계 |
| **70-79%** | 보통 | 해당 단어 재학습 후 재시험 |
| **70% 미만** | 부족 | 단어 학습부터 다시 시작 |
